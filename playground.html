User Test: Risk Of Obsolescence

This piece uses a nostalgic Windows XP-style interface to contrast early internet optimism with today’s AI-driven logic. It reframes planned obsolescence as something that may also be applied to people: our choices, labor, and identities.

Built with HTML, the experience includes a working embedded Minesweeper, an audio player, and a paint section. The project folder also contains the gallery text and the full written description of the piece.

  • The test starts familiar and playful, then gradually corrects and overrides user input.
  • It mirrors recommendation systems that infer identity from patterns and behavior.
  • It questions autonomy: if choices are highly predictable, how free are they?
  • It positions human creativity as lived, embodied, and non-automatable.
  • It asks for pause and resistance against total optimization.
